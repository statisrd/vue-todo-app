# vue-todo-app

Установка зависимостей
``` bash
npm install
или
yarn
```

Установка препроцессора sass-embedded

``` bash
 npm install -D sass-embedded
# или
yarn add -D sass-embedded
```

Запуск в режиме разработки

``` bash
npm run dev
# или
yarn dev
```

Сборка проекта

``` bash
npm run build
# или
yarn build
```
